## Prerequisites
- Node.js >= v9.4.0
- npm ^5.6.0



## Usage

1. Clone this repository
    ```
    $ git clone https://github.com/zfaty/hf_shops_app.git app
    $ cd app
    ```

2. install packages
    ```
      $ npm install
    ```

4. Rename config file and then edit the file. (`.env.example` to `.env`)
    ```
    REACT_APP_API_URL='http://localhost:8000'

    ```

5. Start server
    ```
      $ npm start
    ```
